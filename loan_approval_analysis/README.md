# 🏦 Loan Approval Analysis

This project explores a loan approval dataset to uncover patterns and factors that influence loan approval decisions. Using Python and its data analysis libraries, the dataset is cleaned, analyzed, and visualized to extract meaningful insights.

---

## 📌 Project Objective

The objective of this project is to perform exploratory data analysis (EDA) on loan application data and identify relationships between applicant characteristics and loan approval outcomes.

---

## 🛠️ Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Analysis Performed

### Data Exploration
- Loaded and inspected the dataset
- Examined data types and dataset structure
- Generated descriptive statistics

### Data Cleaning
- Checked for missing values
- Handled inconsistencies in the dataset
- Prepared data for analysis

### Exploratory Data Analysis
- Univariate Analysis
- Bivariate Analysis
- Correlation Analysis
- Distribution Analysis

### Data Visualization
- Count Plots
- Histograms
- Box Plots
- Bar Charts
- Heatmaps
